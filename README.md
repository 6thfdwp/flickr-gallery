flickr-gallery
==============

A Backbone image gallery using flickr search api

### Requirements
==============
jQuery v1.10.0   
Bootstrap v3.3.0   
[Bootstrap image gallery](https://github.com/blueimp/Bootstrap-image-Gallery)   
Underscore.js   
Backbone   

### Features
==============
Search by tags   
Infinite photo stream   
lightbox slides show
